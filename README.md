# Server Performance Stats

A simple Bash script to display basic server performance statistics on a Linux system.

This project helps understand how to monitor CPU usage, memory usage, disk usage, and running processes using standard Linux commands.

---

##  Features

- CPU usage overview
- Memory usage (used and free)
- Disk usage of mounted filesystems
- Top CPU-consuming processes

---

##  Technologies Used

- Bash scripting
- Linux system commands:
  - `top`
  - `free`
  - `df`
  - `ps`
